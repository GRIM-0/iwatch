# iwatch
Frontend: index.php, movies.php, tv-series.php, search.php, moviedetails.php, seriesdetails.php, favorites.php, watchlist.php, profile.php render UI with navbar.php and modals.php.

Backend: auth.php, signup_handler.php, signin_handler.php, save_preferences.php, favorites_handler.php, reviews_handler.php (and likely watchlist_handler.php) process requests.

Config: config.php ties everything with database and API access.

JS/CSS: script.js and styles.css enhance interactivity and look.
